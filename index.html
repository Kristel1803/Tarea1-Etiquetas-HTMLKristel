Listas Ordenadas

Primeras etiquetas para emprezar tu página web
  1. <!DOCTYPE>: Declaración del tipo de documento
  2. <html>: Raíz del documento HTML
  3. <head>: Contenedor de metadatos (como título, enlaces a CSS, etc.)
  4. <title>: Título de la página (visible en la pestaña del navegador)
  5. <meta>: Metadatos (información sobre el documento, como el juego de caracteres)
  6. <link>: Enlace a recursos externos (como hojas de estilo)
  7. <script>: Código JavaScript
  8. <body>: Cuerpo del documento (contenido visible)

Listas Desordenadas

Etiquetas HTML que puedes usar sin un orden específico
  • <h1>- <h6>: Encabezados de diferentes niveles
  • <p>: Párrafo
  • <a>: Enlace(anchor)
  • <img>: Imagen
  • <span>: Contenedor en linea
  • <header>: Cabecera de una sección o página
  • <footer>: Pie de una sección o página

Lista de Descripción

Descripción breve de algunas etiquetas HTML comunes
<html>
Define la estructura principal de un documento HTML.
Es el elemento raíz de la página.
<head>
Contiene información meta sobre el documento,
como el título y enlaces a archivos CSS o JavaScript.
<body>
Define el contenido visible del documento, como texto, imágenes, enlaces, etc.
<h1> - <h6>
Define encabezados con diferentes niveles de importancia,
de <h1> (más importante) a <h6> (menos importante).
<p>
Representa un párrafo de texto.
<a>
Crea un enlace a otra página web o recurso externo.
<img>
Incorpora una imagen en la página web.
<ul>
Define una lista desordenada, donde los elementos se presentan con viñetas.
<ol>
Define una lista ordenada, donde los elementos se presentan con números o letras.
<li>
Define un elemento dentro de una lista (ya sea desordenada o ordenada).
<div>
Elemento de bloque que agrupa contenido y se usa principalmente para dar estructura o aplicar estilos.
<span>
Elemento en línea que sirve para aplicar estilos o manipular texto o pequeños bloques de contenido.
<form>
Define un formulario para capturar datos del usuario.
<input>
Crea un campo de entrada para que el usuario introduzca datos.
<button>
Crea un botón interactivo en la página web.

Información Adicional sobre Elementos HTML

Haz clic para explorar mas sobre los elementos HTML

<script>: Inserta un segmento de codigo JavaScript en tu documento HTML.
Esto permite añadir funcionalidades interactivas a tu sitio web.

<form>: Reúne diversos controles de entrada para enviar datos a un servidor.
Los controles pueden incluir campos de texto, botones de selección, casillas de verificación y más.

<button>: Define un bóton que los usuarios pueden hacer clic para ejecutat una acción específica,
como enviar un formulario o invocar una función JavaScript.
